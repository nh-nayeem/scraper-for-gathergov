# GatherGov Web Scraper Project

## Setup Requirements

### Prerequisites
- Python 3.7+
- Playwright
- BeautifulSoup4
- pip (Python package manager)

### Installation
1. Install the required Python packages:
   ```bash
   pip install playwright beautifulsoup4
   ```

2. Install browser binaries for Playwright:
   ```bash
   playwright install
   ```
   This will download the necessary browser binaries for Chromium, Firefox, and WebKit.

## Project Structure

Each task is contained in its own folder with a descriptive name (e.g., `task1/`, `task2/`). Each task folder contains:
- `README.md` - Task-specific instructions and documentation
- `scraper.py` - The main script to run
- `data/` - Directory containing input/output files

## Running Tasks
Navigate to the specific task directory and follow the instructions in its README.md file for execution details.

All tasks are in the respective folder as named. 
Instructions are in the README.md file in the corresponding folders.
